Stand de fruits et légumes - Mickaëlla GRONDIN-VERDON (2021)
===========================================
Cet unity package a été créé dans le but d'y placer un agent conversationnel animé (GRETA). 

Pour le réaliser, j'ai utilisé différents assets gratuits trouvés sur les sites suivants : 
- Asset Store de Unity
- Sketchfab
- Turbosquid

Je n'ai fait que placer les éléments afin d'obtenir le rendu actuel. 
J'ai réalisé deux types de stands qui se distinguent par la position des cagettes 
(changement de l'étagère entre les préfabriqués standV1 et standV2). 

Les préfabriqués roomAgent sont les utilisations du standv2 dans le cadre de notre protocole. 
Lors que la scène est run et que vous êtes en game view, vous pouvez changer l'affichage grâce au script changement
avec la barre espace. Un script permet de de faire les changements d'affichage via le bouton Move du projet TouchDesigner associé
au protocole. Les différents affichages sont les suivants : 
- 1 boule rouge qui passe par toutes les cagettes puis sur la position de la tête de l'agent à la suite (permet le calibrage)
- Des produits dans les cagettes, leur nom et leur prix. 
- 1 liste de course qui change à chaque fois. 

Certains materials ne sont pas déclarés et peuvent être fait par vous même en utilisant les textures.

==============
Ce projet peut comporter des incohérences ou être très brouillon pour toute personne qui possède des connaissances avancée sur Unity (ce qui n'est pas mon cas)
